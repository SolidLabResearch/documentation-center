# SolidLab Documentation Center

## Usage

### Download data

1. `cd data`
2. `npm i`
3. `npm start`

### Serve website

1. Install [mkdocs](https://www.mkdocs.org/)
2. `mkdocs serve`

### Update GitHub pages

1. `mkdocs gh-deploy`
